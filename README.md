# SplitPay

Automated payment splitting contract.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/SplitPay.sol`
- Stacks: `contracts/stacks/contracts/splitpay.clar`

## Setup
```bash
pnpm install
pnpm dev
```
